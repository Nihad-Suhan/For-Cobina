# Cobina's Tiny Bird Dance Floor

A self-contained browser surprise for Cobina: animated pet birds, a stylized dancing cameo, Strawberry Wine-inspired shimmer, and a local MP3 backing track.

Open `index.html` in a browser and tap **Tap to begin music**. Browsers require one tap before sound can play.

## Publishing On GitHub Pages

Make sure these files are uploaded together:

- `index.html`
- `styles.css`
- `script.js`
- `assets/strawberry-wine.mp3`

The folder must be named exactly `assets`, and the MP3 must be named exactly `strawberry-wine.mp3`, all lowercase. GitHub Pages paths are case-sensitive.
